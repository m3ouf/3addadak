سياسة الخصوصية — عدادك

آخر تحديث: مارس ٢٠٢٦

مقدمة

تطبيق "عدادك" بيحترم خصوصيتك تماماً. التطبيق ده مصمم إنه يشتغل بدون ما يجمع أو يرفع أي بيانات شخصية.

البيانات اللي بنخزنها

التطبيق بيخزن البيانات دي على جهازك بس (AsyncStorage):

  • بيانات العربية: الماركة، الموديل، سنة الصنع، قراية العداد
  • سجل الصيانة: تواريخ الصيانة، التكاليف، الملاحظات
  • إعدادات التنبيهات: مواعيد التنبيه لكل قطعة
  • بيانات GPS: المسافات المتتبعة (لو فعّلت التتبع)

البيانات اللي مش بنجمعها

  • مفيش سيرفر: مفيش أي بيانات بتتبعت لأي سيرفر
  • مفيش تحليلات: مفيش Google Analytics أو أي أداة تتبع
  • مفيش إعلانات: مفيش أي SDK إعلانات
  • مفيش تسجيل دخول: مش محتاج حساب أو إيميل
  • مفيش مشاركة: مفيش أي بيانات بتتشارك مع أي طرف تالت

صلاحيات التطبيق

الموقع (اختياري)

  • الغرض: حساب المسافة اللي بتمشيها بالعربية تلقائي
  • متى: بس لما تفعّل التتبع بالـ GPS
  • الفلترة: التطبيق بيتجاهل السرعات أقل من ١٥ كم/ساعة
  • التخزين: بيانات المسافة بتتخزن على جهازك بس
  • الخلفية: التتبع في الخلفية بيستخدم foreground service عشان يفضل شغال وأنت سايق

الإشعارات (اختياري)

  • الغرض: تنبيهك لما قطعة تقرب من موعد الصيانة
  • التحكم: تقدر تفعّل/تعطّل التنبيهات لكل قطعة لوحدها

الكاميرا (اختياري)

  • الغرض: تصوير عداد العربية وقراية الرقم تلقائي (OCR)
  • المعالجة: الصورة بتتعالج على جهازك باستخدام ML Kit — مش بتترفع في أي مكان
  • التخزين: الصور مش بتتخزن — بتتمسح بعد القراية

أمان البيانات

  • كل البيانات مخزنة محلياً على جهازك باستخدام AsyncStorage
  • لو مسحت التطبيق، كل البيانات هتتمسح معاه
  • مفيش نسخة احتياطية على السحابة (حالياً)

حقوقك

  • الحذف: تقدر تحذف أي عربية أو قطعة أو سجل صيانة في أي وقت
  • التحكم: كل الصلاحيات اختيارية وتقدر تعطّلها
  • الشفافية: كود التطبيق مفتوح المصدر على GitHub
