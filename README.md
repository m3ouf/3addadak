# عدادك — 3addadak

**تطبيق متابعة صيانة العربية**

تطبيق موبايل بيساعدك تتابع صيانة عربيتك بسهولة. بيفكرك قبل ما أي قطعة تحتاج تتغير.

---

## المميزات

- **62 قطعة في 10 أقسام** — زيوت، فلاتر، فرامل، كاوتش، كهرباء، تكييف، وأكتر
- **تنبيهات ذكية على 3 مستويات** — تمام / قربت / فاتت عليها
- **تتبع بالـ GPS** — بيحسب الكيلومترات تلقائي وأنت سايق (سرعة > 15 كم/س)
- **أكتر من عربية** — تابع صيانة كل عربياتك في مكان واحد
- **اختيار ماركة وموديل** — 26 ماركة من السوق المصري جاهزين
- **قطع مخصصة** — ضيف أي قطعة مش موجودة في القايمة
- **تعديل فترة الصيانة** — غيّر الكيلومترات لكل قطعة حسب عربيتك
- **واجهة عربي كاملة** — RTL، خط Cairo، لهجة مصري

## التقنيات

| التقنية | الاستخدام |
|---------|-----------|
| Expo SDK 55 | إطار العمل |
| expo-router 55.0.5 | التنقل بين الصفحات |
| TypeScript | لغة البرمجة |
| React Native Reanimated | الحركات |
| expo-location | تتبع GPS |
| expo-notifications | التنبيهات |
| AsyncStorage | تخزين البيانات محلياً |

## هيكل المشروع

```
3addadak/
├── app/                    # صفحات التطبيق (expo-router)
│   ├── (tabs)/             # التابات الرئيسية
│   │   ├── index.tsx       # الرئيسية
│   │   ├── alerts.tsx      # التنبيهات
│   │   └── settings.tsx    # الإعدادات
│   ├── car/                # صفحات العربية
│   │   ├── add.tsx         # إضافة عربية
│   │   └── [id]/           # تفاصيل العربية
│   │       ├── index.tsx   # تفاصيل
│   │       ├── edit.tsx    # تعديل
│   │       ├── update-km.tsx
│   │       ├── custom-part.tsx
│   │       ├── gps-trips.tsx
│   │       └── part/[partType].tsx
│   └── welcome.tsx         # شاشة الترحيب
├── src/
│   ├── components/         # المكونات
│   │   ├── car/            # مكونات العربية
│   │   ├── parts/          # مكونات القطع
│   │   ├── layout/         # Header, TabBar, ScreenWrapper
│   │   └── ui/             # GlassCard, GradientButton, PickerModal...
│   ├── constants/          # الثوابت
│   │   ├── parts.ts        # 62 قطعة
│   │   ├── car-brands.ts   # 26 ماركة
│   │   ├── strings.ts      # النصوص
│   │   └── colors.ts
│   ├── hooks/              # React hooks
│   ├── services/           # حسابات، GPS، تخزين، إشعارات
│   ├── store/              # Context + Reducer
│   └── utils/              # أدوات مساعدة
└── assets/                 # صور وخطوط
```
